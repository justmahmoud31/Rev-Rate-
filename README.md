## Installation

Follow these steps to set up the project on your local machine:

1. **Clone the repository:**
   ```sh
   git clone https://github.com/justmahmoud31/Rev-Rate-.git
   ```
2. **Open the project :**
   ```sh
    cd Rev-N-Rate 
    ```
3. **Install The dependancies** 
   ```sh 
    npm i 
    ```
4. **Run The project**
#### Using Nodemon 
   ```sh
    nodemon 
   ```
#### Using npm
  ```sh
    npm start
   ```
